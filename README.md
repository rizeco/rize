## Novidades nesta versão
- Fontes: Poppins + DM Mono + Space Mono
- Cards flip animados na seção The Brand
- "Quem sabe, sabe." (traduzido)
- Badge do hero corrigida (z-index fixado)

1. Coloque as imagens na pasta `assets/images/`
2. Em `js/main.js` linha 4, troque o número do WhatsApp
3. Abra `index.html` no navegador
